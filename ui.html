Accessibility Testing

Features

  • Feature 1 ➤ Text Readability: Analyze text layers for readability metrics, such as font size, line height, and letter spacing.
  • Feature 2 ➤ Color Contrast: check the color contrast ratios between text and background colors. Will probably use color contrast library like axe-core to implement this one.
  • Feature 3 ➤ Heading hierarchy: Check if the design has order of hierarchy for headings.
  • Feature 4 ➤ Link Accessibility: check links have discernible text and provide additional context for screen reader users.
